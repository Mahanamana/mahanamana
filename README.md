# Customize Start Bootstrap - Resume

[Resume](https://startbootstrap.com/template-overviews/resume/) is a resume and CV theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/). This theme features a fixed sidebar with content sections to build a simple, yet elegant resume.
Our contribution is level of responsive and redesign for example addition of bootstrap grids for a better arranging to the different device, modification of the color themes and font-size enhancement.
